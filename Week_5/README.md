This practicum aims to help students understand the basic structure of a Flutter project, layout configuration, and the implementation of basic widget components. This practicum covers the Text widget for displaying an introductory sentence with customized text styling, as well as the Image widget for displaying the Polinema logo from the assets directory.

This practicum also uses a FloatingActionButton with a thumbs-up icon, along with an AlertDialog pop-up function that displays an interactive message when the button is pressed. Additionally, an input element in the form of a TextField labeled “Name” is added, along with a call to showDatePicker to display an interactive calendar for selecting a date. All these visual components are managed and neatly arranged within the main layout using the Scaffold framework.

The application was created to demonstrate the implementation of basic user interface components, ranging from the use of static text, images, buttons, dialogs, inputs, to date pickers. Through this exercise, it can be understood that an organized widget tree structure and the proper use of Scaffold make UI code cleaner, easier to understand, and highly interactive for mobile app development needs.
